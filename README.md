# Personalized Book Learning Agent

## Overview
This project is an AI-powered learning agent that helps users learn from books by retrieving relevant knowledge using semantic search and maintaining long-term memory of user understanding.

The goal is to make knowledge in books more accessible to people who struggle with reading habits or time constraints.

## Core Features
- Semantic search over book content
- Personalized explanations based on user context
- Long-term memory using a vector database (Qdrant)

## Tech Stack
- Python
- Qdrant (Vector Database)
- Embeddings (to be added)
- CLI-based interaction
